## 1. ゲーム概要

| 項目 | 内容 |
|------|------|
| ジャンル | 横スクロールアクション（エンドレスラン系） |
| プラットフォーム | Web（PWA）、iOS/Android予定（Capacitor） |
| 対応言語 | 日本語 / 英語 |
| 画面方向 | 横画面（ランドスケープ）固定 |
| 操作方法 | タッチ操作（モバイル） / キーボード操作（PC） |
| 解像度 | 820 × 450（内部キャンバス） |
| フレームレート | 60 FPS（固定タイムステップ） |
| レンダリング | Canvas 2D（ピクセルアート調） |
| データ保存 | localStorage |

### コンセプト
プレイヤーキャラクター「ぴよ氏」を操作し、自動スクロールするステージを走り抜ける。敵を踏んで倒し、コインを集め、パワーアップアイテムを活用しながら、2400mごとに出現するボスを撃破してラウンドを進める。

## 3. 操作方法

### 3.1 タッチ操作（モバイル）

| エリア | 位置 | 操作 |
|--------|------|------|
| 左移動 | 画面左端 0-18% | タッチ中に左へ移動 |
| 右移動 | 画面左端 18-33% | タッチ中に右へ移動 |
| デッドゾーン | 画面中央 33-70% | 操作なし（誤タップ防止） |
| ジャンプ | 画面右端 70-100% | タッチでジャンプ |

**ジェスチャー:**
| ジェスチャー | 動作 |
|--------------|------|
| 下スワイプ（15px以上 / 500ms以内） | 足場をすり抜ける |
| 上スワイプ（-20px / 500ms以内） | ショップに入る（建物付近にいれば画面のどこでも検出） |

### 3.2 キーボード操作（PC）

| キー | 動作 |
|------|------|
| ← / A | 左移動 |
| → / D | 右移動 |
| Space / ↑ / W | ジャンプ |
| ↓ / S | 足場すり抜け |
| Escape / P | ポーズ |

| 攻撃 | パラメータ |
|------|----------|

| 卵攻撃 | 拡散角度±0.3rad、弾速4px/f、重力+0.15px/f²、最大寿命300F |
| 火炎放射 | 弾速(5+乱数×2)px/f、弾サイズ24×24px、10Fごと発射 |
| ジャンプ踏み | velY=-14、重力で落下、地面着地で終了 |
| 閃光アタック | チャージ後に閃光30F、地面近く(GROUND_Y-70以内)のプレイヤーにダメージ |

### 6.6 ボスへのダメージ方法

| 方法 | ダメージ | 備考 |
|------|---------|------|
| 踏みつけ（ストンプ） | 1 HP | プレイヤーがボスの上30%に着地、+500点 |
| エナジー弾 | 0.2 HP/発 | 5発で1HP相当 |

### 6.7 怒りモード
- ストンプ命中後に発動（2.5秒間 = 150F）
- 移動速度が2倍に
- 連続ストンプ不可（クールダウン90フレーム）
- 突進・ジャンプ等の実行中攻撃をキャンセル

## 7. ステージ構成

### 7.1 自動スクロール

| パラメータ | 値 |
|-----------|-----|
| 基本スクロール速度 | 1.2 px/frame |
| 速度上昇間隔 | 300mごとに+20% |
| 最大速度 | 基本の5倍（6.0 px/frame） |
| 安全地帯速度制限 | 1.5 px/frame固定上限 |
| ボス戦中 | スクロール停止 |

### 7.2 バイオームシステム

2400mの周期で4つのバイオームが循環（各600m）:

| インデックス | バイオーム | 地形タイプ | 特殊効果 |
|------------|-----------|-----------|---------|
| 0 | 草原 | 通常 | なし（穴あり） |
| 1 | 砂漠 | 流砂 | 移動速度低下、穴なし |
| 2 | 雪山 | 氷 | 慣性増加（滑る） |
| 3 | 夜 | 暗い地形 | rgba(5,5,30,0.35)の夜間オーバーレイ、星空演出 |

- バイオーム間はグラデーションで滑らかに遷移（約2秒間）

### 7.3 地形

| 地形タイプ | 説明 |
|-----------|------|
| ground | 通常の地面 |
| hole | 穴（落下でダメージ） |
| elevated | 高台（60-100px） |
| ice_elevated | 氷の高台 |
| quicksand_elevated | 流砂の高台 |

- 穴出現確率: 15%（砂漠では穴なし、高速時は `max(0, 0.15 × (5.0 - speedMult))` に減少）
- 安全地帯（ボス手前250m）: 平坦地形のみ

### 7.4 足場（プラットフォーム）

出現間隔: 120フレームごとに60%の確率

**基本タイプ:**
| タイプ | 出現確率 |
|--------|---------|
| 雲 (cloud) | 67% |
| 浮遊地面 (floating_ground) | 33% |

**特殊足場（20%の確率で付与）:**

| タイプ | 確率 | 効果 |
|--------|------|------|
| 動く足場 (moving) | 8% | 上下に振動（振幅40-70px） |
| 消える足場 (disappearing) | 6% | 乗ると1.5秒後に消滅 |
| バネ足場 (spring) | 6% | 乗ると通常の1.4倍の高さでジャンプ |

- サイズ: 幅120-200px × 高さ30px
- 出現高度: Y 164-286px

## 20. gameRound制御一覧

`gameRound` は1から始まり、ボス撃破後のラウンド移行時に+1される。

| 影響項目 | 計算式 / 条件 |
|---------|-------------|
| ボス出現距離 | 2400 × gameRound (m) |
| ボスHP | 10 + max(0, gameRound - 3) × 3 |
| ステージBGM | (gameRound - 1) % 3 → 0:stage / 1:stage2 / 2:stage3 |
| 敵速度スケール | × (1 + (gameRound - 1) × 0.3) |
| ボス後スクロール速度 | savedSpeed × (1 + (gameRound - 1) × 0.2)、上限6.0px/f |
| 地上ミニオン（ボス戦） | ラウンド2以降、間隔 max(90, 180 - (gameRound - 2) × 20)F |
| 飛行ミニオン（ボス戦） | ラウンド3以降、間隔 max(120, 240 - (gameRound - 3) × 20)F |
| ボスアリーナアイテム | ラウンド3以降はドロップ停止 |
| 閃光アタック | ラウンド2以降で解禁 |
