## Phase 1: 土管の入場性改善 — ✅完了（v1.407）

**問題:** 入口土管は幅72px（プレイヤー約44px）。高速域では「狭い足場に着地→下スワイプ」が間に合わず、入場がほぼ運になる。

**施策（セットで実施）:**
- [x] **1-a. 土管タイム（本命）**: プレイヤーが土管の上に乗った瞬間、世界のスクロールを大幅減速（×0.2〜0.3）。入場する・土管から離れる・最大2秒経過のいずれかで解除。1つの土管につき1回だけ発動。
  - 実装メモ: `updateGameSpeed()` が毎tick速度を再計算するため、`pipeAssistTimer` を設けて算出後に乗算する形にする（直接gameSpeedを書くと次tickで戻る）。
- [x] **1-b. 入場判定の寛容化**: 真下判定（`findPlatformUnder`）に加え、接地中かつ土管と水平±12px重なりでも入場可に。
- [x] **1-c. 幅の拡大**: PIPE_W 72→88px → **1.409でユーザー指定により176px（倍幅）**。
- [x] **1-d. ヒント強化**: 既存の「下向き矢印」に「▼したにスワイプ」の小テキストを添える＋土管タイム中は点滅。

**受け入れ基準:** 最高速度域でも、初見プレイヤーが狙えば安定して入場できる（土管タイム発動中に落ち着いてスワイプできる）。

## Phase 2: マリオ風の土管 出入り演出 — ✅完了（v1.408）

**入場演出:** スワイプ確定 → 入力ロック＋無敵 → ぴよ氏を土管中央へスナップ（約0.15秒）→ 下降して土管に消える（約0.5秒）→ SE → 部屋へ遷移。
**退場演出:** 出口で右押しホールド（1.410で42f→21f=0.35秒・ゲージ表示は撤去）→ 部屋側で横土管の口へ歩き込んで消える（クリップ方式）→ 本編へ復帰 → **保存位置の土管から上昇して出てくる** → 操作復帰。
（世界は部屋滞在中凍結済みなので、復帰時に入口土管は必ず足元にある＝savedPlayer位置）

- [x] 状態機 `pipeRoomState.anim = 'none'|'in'|'outRoom'|'outWorld'` ＋タイマー。演出中は入力・被弾・多重入場なし
- [x] **描画方式**: 当初のZ順再描画は「裏に回った」ように見えたため、**1.409で「穴の手前縁より上だけ描く」クリップ方式に変更**（PIPE_MOUTH_LINE=5px・スプライト実測）＝穴に沈む/出てくる見た目
- [x] SE「シュポッ」（オシレータ合成で実装＝追加音源不要）: 新規音源が無ければ SoundManager のオシレータで下降スイープを合成（追加ファイル不要）。ユーザー支給音源があればそれを使用（**Sunoならタグ除去ルール適用**）
- [x] 回帰確認: Android戻る（1.405の履歴push/pop）・縦画面ポーズ・必殺技ガードと干渉しないこと

**受け入れ基準:** 入退場ともぴよ氏が土管に消える/現れる。演出中に死亡・被弾・二重入場が発生しない。

## 📌 序盤圧縮 詳細起案（Phase 3の選定材料）

### 案A: 初回ラン圧縮スクリプト（推奨・規模:小）
初回ラン（`achievementStats.totalPlays === 0`）だけ、世界の見どころを前倒しする。2回目からは通常。
- **A-1**: ボス出現距離を半分に（1000m → 500m）— 初回5分で必ずボス戦を体験
- **A-2**: 土管（ボーナス部屋）を200〜400mに必ず配置（`pickPipeTargetDist` に初回分岐）— Phase1/2の改善が初回から輝く
- **A-3**: ショップ建物も前倒し — 買う楽しみの提示
- リスク: 低（初回のみの分岐。既存バランス不変）

### 案B: おためしストック＋ポーチ予告（推奨・規模:小）
- **B-1**: 初回ランのみ、開始時にストックへ「バリア」1個を配布＋トースト「ストックはタップでつかえる！」— 本作最大の武器（ストック→ポーチ）の学習
- **B-2**: 初回ゲームオーバー画面に予告カード「まほうのポーチ＝金枠でアイテムを毎回持ち込める」— 継続動機の注入
- リスク: 低（バリア1個・初回のみ。経済影響なし）

### 案C: ティザー演出（規模:中・優先度低）
初回スタート前にボス5体のシルエット等のカードを見せる（スキップ可）。
- 懸念: 開始までの時間が延びる＝「圧縮」と逆行。**A/Bで実物を見せる方が強い**ため、私は見送りを推奨。

### 案D: ミッション進捗のHUDトースト（規模:小〜中）
デイリーミッションが進んだ瞬間に「ミッション: あと3体！」のミニトーストを右上に出す。序盤限定ではないが「走る理由の常時可視化」として効く。
- リスク: 表示頻度の調整が要る（うるさくならないように間引き）
