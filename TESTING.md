# ぴよ氏の冒険 — 回帰テストチェックリスト

**用途:** リファクタリングの各フェーズ完了後、このリストを上から順に実機（スマホ）で確認する。
1つでも「いつもと違う」動きがあれば、そのフェーズのPRを疑う。

**確認環境:** スマホ実機（PWA推奨）。PC確認項目には 💻 マークあり。
**所要時間目安:** 全項目 約20〜30分

---

## 0. 起動・タイトル画面

- [ ] スプラッシュ画面（注意書き＋クレジット）が表示され、タップでタイトルへ進む
- [ ] タイトルBGMが鳴る
- [ ] バージョン番号（左下）が最新であること ← **テスト開始前に必ず確認。古い場合はUPDATEボタンで更新**
- [ ] 右上「↻ UPDATE」ボタンが押せて、ページが再読み込みされる（ゲームが始まってしまわないこと）
- [ ] SETTING / RANKING / SHOP ボタンがそれぞれ正しい画面を開く
- [ ] ボタン以外の場所をタップするとゲームが開始される
- [ ] （日付が変わって初回の場合）ログインボーナスのポップアップが出て、報酬が貯金に加算される

## 1. 基本操作

- [ ] 画面左側エリアのタッチで左移動・右移動が切り替わる（下部バーのL/Rが光る）
- [ ] 画面右側エリアのタップでジャンプ（下部バーJumpが光る）
- [ ] 足場の上で下スワイプ → 足場をすり抜けて落ちる
- [ ] 中央のデッドゾーンをタップしても何も起きない
- [ ] 💻 PCで矢印キー/WASD/スペースの移動・ジャンプが効く
- [ ] ポーズボタン（右上）でポーズ → 再開 / リタイアの両方が機能する
- [ ] ポーズ中に画面タップで再開できる

## 2. アイテム・パワーアップ

- [ ] コインを取るとスコアが150円（×ボーナス）増え、効果音が鳴る
- [ ] ハート: HP+1（HP満タン時は+1000円）
- [ ] レモン缶: 緑オーラが出てジャンプが高くなる（約5秒）
- [ ] シールド: 青オーラ。敵に触れると敵が倒れ、ダメージを受けない（約5秒）
- [ ] エナジー: 赤オーラ。自動で弾が発射される（約8秒）
- [ ] マグネット: 紫オーラ。近くのコインが吸い寄せられる（約10秒）
- [ ] パワーアップ中、HUDに残り時間バーが表示される

## 3. 敵・コンボ

- [ ] 地上敵（ひよこ）を踏むと倒せて+200円、軽くバウンドする
- [ ] 飛行敵を踏むと倒せて+300円
- [ ] 敵に横から触れるとHP-1、赤フラッシュ＋約3秒無敵（点滅）
- [ ] 連続で敵を倒すとコンボ表示（2体目以降）が出てスコア倍率が上がる
- [ ] 5コンボで画面シェイク＋フラッシュ
- [ ] 穴に落ちるとHP-1で上空から復帰（ゆっくり降りてくる）

## 4. バイオーム・ステージ

- [ ] 600mごとに背景が変わる（草原→砂漠→雪山→夜）
- [ ] 砂漠: 移動が遅くなる（流砂）
- [ ] 雪山: 足が滑る（氷）
- [ ] 夜: 画面が暗くなり星が見える
- [ ] 300mごとに「SPEED UP!」表示が出て速くなる
- [ ] 動く足場に乗ると一緒に上下し、地面にめり込まない
- [ ] 消える足場は乗ってから約1.5秒で消える
- [ ] バネ足場で高くジャンプできる

## 5. ステージショップ（ボス手前）

- [ ] ボス出現の手前で減速し、ショップ建物が出現する
- [ ] 建物付近で「↑スワイプで入店」の表示が出る
- [ ] **画面のどこで上スワイプしても**入店できる（左エリア・右エリア・中央すべて）
- [ ] 入店するとショップBGMに切り替わる
- [ ] 「かいものする」: 商品一覧が出る。1回目タップでセリフ、2回目で購入確認（はい/いいえ）
- [ ] 購入するとお金が減り、ストック品はストックスロットに入る
- [ ] お金不足のとき購入できず、店員が断るセリフを言う
- [ ] 「ものをうる」: ストック品を購入価格の半額で売れる
- [ ] 「貯金する」: 所持金の50%が貯金に移る（**金額を電卓で検算**）。1回の入店で1回だけ
- [ ] 「広告を見てボーナス」: 所持金の30%（100〜3000円）が増える
- [ ] 店員のセリフが「です/ます」調であること（「〜なのだ」になっていないこと）
- [ ] 「お店を出る」で退店し、ゲームが再開される
- [ ] 退店後、同じショップに再入店できない

## 6. ボス戦

- [ ] 2400m×ラウンド数でWARNING演出＋警告音（約2秒で音が止まる）
- [ ] ボスが右から歩いて登場し、ボスBGMに切り替わる
- [ ] ボスを踏むとダメージ（+500円）、ボスが点滅して怒りモード（赤く速くなる）
- [ ] エナジー弾でもボスのHPが減る
- [ ] ボスの体当たり・卵・炎・閃光でHP-1
- [ ] **シールド中はボスの全攻撃（体当たり・卵・炎・閃光）が無効**
- [ ] 閃光攻撃（ラウンド2+）: 溜め演出のあと、ジャンプで回避できる
- [ ] ボス撃破で爆発演出→コイン25枚→「DEFEATED!」→ファンファーレ
- [ ] **撃破の瞬間、画面に残っていた敵が全部消える**
- [ ] 「ROUND 2」表示後、ゲーム再開してBGMが変わる（stage→stage2）

## 7. ゲームオーバー・復活

- [ ] HP0でゲームオーバー画面（距離/金額/撃破数/レベルの4項目表示）
- [ ] 復活薬をストックに持っているとHP2で自動復活（上空から）
- [ ] 「広告を見て復活」: HP3で**上空から**復活。**固まらないこと**。コントロールバーが表示されていること
- [ ] 広告復活は1プレイ1回のみ（2回目のゲームオーバーではボタンが出ない）
- [ ] リトライで最初からやり直せる
- [ ] 「タイトルへ」でタイトルに戻り、タイトルBGMが鳴る

## 8. ランキング・名前入力

- [ ] ハイスコア時に名前入力画面が出る
- [ ] 名前を入れて登録→「ランキングに登録しました！」＋順位表示
- [ ] 「かすみ」が登録できる（NGワード誤判定しない）
- [ ] 不適切な名前（例:「ばか」）は登録できない
- [ ] スキップでも記録される（名無しのぴよ氏）
- [ ] ランキング画面: 金額/距離/撃破/レベルの4タブが切り替わる

## 9. 設定・その他

- [ ] サウンドOFF→全ての音が止まる。ON→復帰
- [ ] 言語をEnglishに切替→UIが英語になる（ガイドのスクロール表示「▼ Scroll」も）
- [ ] 日本語に戻せる
- [ ] 遊び方ガイドが開け、最後までスクロールできる（途中で文字サイズが変わらないこと）
- [ ] プライバシーポリシー/利用規約リンクが開く
- [ ] 💻 Android実機: 戻るボタンで各画面が正しく閉じる（設定/ランキング/ガイド/ショップ）

## 10. PWA・データ保存

- [ ] ゲームを完全に閉じて再起動→貯金・アップグレード・言語設定が残っている
- [ ] 機内モードで起動→ゲームが動き、BGM・効果音（選択音含む）が鳴る
- [ ] タイトルショップで購入したアップグレードが次のプレイに反映される（例: タフネス→初期HP増）

## 金額検証メモ（フェーズ4で特に重要）

| 項目 | 期待値 |
|------|--------|
| 貯金額 | 所持金の50%（端数切り捨て） |
| 売却額 | 購入価格の50%（端数切り捨て） |
| ショップ広告ボーナス | 所持金の30%、最低100・最大3000 |
| タイトルショップ広告 | 固定3000円、クールダウン4時間 |
